# cher-llm-methodology

Dieses Repository dokumentiert eine strukturierte Methodik zur Zusammenarbeit mit Large Language Models (LLMs) wie ChatGPT in komplexen, mehrschrittigen Projekten.

Ziel ist es, einen Rahmen zu schaffen, in dem:
- Kontext über viele Chat-Sitzungen hinweg stabil bleibt,
- Erkenntnisse nicht verwässern oder verloren gehen,
- Ergebnisse nachvollziehbar dokumentiert werden,
- und die Zusammenarbeit mit dem LLM Schritt für Schritt verbessert werden kann.

## Mission und Scope

Die grundlegende Motivation, Problemstellung und Zielsetzung des Projekts sind in folgendem Dokument beschrieben:

## Dokumentationsstruktur (erste Version)

Geplant ist eine schrittweise Erweiterung des `docs/`-Verzeichnisses, unter anderem um:

- methodische Grundlagen (z. B. Umgang mit ChatGPT-Projekten),
- konkrete Workflows und Checklisten,
- Beispiele aus realen Projekten (z. B. cher-webapp-core),
- wiederverwendbare Prompts und Templates.

Dieses Repository wird iterativ aufgebaut und dient als zentrale, versionierbare Wissensbasis für die eigene LLM-gestützte Arbeitsweise.
